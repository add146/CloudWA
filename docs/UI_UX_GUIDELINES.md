# UI/UX Design Guidelines

## Overview

Panduan desain UI/UX untuk CloudWA Flow, mencakup prinsip desain, color palette, komponen, dan saran per target user.

---

## 1. Prinsip Desain

| Prinsip | Implementasi |
|---------|--------------|
| **Simplicity First** | Dashboard minimalis, fitur advanced di menu lanjutan |
| **Progressive Disclosure** | Info dasar dulu, detail muncul saat dibutuhkan |
| **Visual Feedback** | Status connection real-time, progress bar jelas |
| **Mobile Friendly** | Dashboard bisa diakses dari HP |
| **Dark Mode** | Toggle dark/light tersedia |

---

```css
:root {
```
